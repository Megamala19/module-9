# mailregisterapp

React Hands-On Lab: Forms & Validation.

A `register.js` component that renders a Name / Email / Password form and
validates each field on change (`onChange`) and on submit (`onSubmit`):

- Name: at least 5 characters
- Email: must contain `@` and `.`
- Password: at least 8 characters

## Run it

```bash
npm install
npm start
```

## Test it

```bash
npm test
```
